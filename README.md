# VPNGate Manager Pro — Web

VPNGate Manager Pro - это веб-приложение для поиска и загрузки .ovpn конфигураций с VPNGate. Приложение позволяет просматривать список серверов, тестировать их доступность, подключаться к ним и управлять настройками.

## Особенности

- Просмотр списка VPN-серверов с фильтрацией по стране и статусу
- Тестирование доступности серверов
- Массовое тестирование всех серверов
- Загрузка .ovpn конфигурационных файлов
- Прямое подключение к VPN (при наличии плагина OpenVPN)
- Поддержка нескольких языков (русский/английский)
- Адаптивный дизайн для разных устройств
- Поддержка пользовательских источников данных

## Улучшения мобильной версии

### 1. Адаптивный дизайн
- Полностью адаптивный интерфейс, который корректно отображается на устройствах с различными размерами экрана
- Поддержка экранов от 320px до 4K
- Оптимизация для тач-устройств

### 2. Улучшенное мобильное взаимодействие
- **Свайп-жесты**: Возможность открывать/закрывать детали сервера свайпом
- **Улучшенные сенсорные цели**: Все кнопки и элементы управления имеют минимальный размер 44px для удобного нажатия
- **Улучшенная навигация**: Логические группы элементов управления для мобильных устройств

### 3. Оптимизированные элементы управления
- **Мобильные элементы управления**: Новые контейнеры `.mobile-controls` и `.mobile-test-controls` для лучшей организации элементов на мобильных устройствах
- **Адаптивные кнопки**: Кнопки "Тестировать все" и "Остановить" теперь корректно отображаются и работают на мобильных устройствах
- **Улучшенная доступность**: Добавлена поддержка клавиатурной навигации и ARIA-атрибутов

### 4. Оптимизация производительности
- **Эффективная обработка событий**: Использование passive событий для сенсорных жестов
- **Оптимизированные стили**: Уменьшение количества перерисовок и улучшение производительности прокрутки

## Установка и запуск

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
3. Приложение готово к использованию

> Примечание: Некоторые функции (например, прямое подключение к VPN) могут требовать специальных плагинов или расширений браузера.
